This folder consists the dataset that is being used for the data analysis in this project.
The 5 files namely 'dataset1.txt', 'dataset2.txt', ... consists of 1 lakh single digit numbers in random order ordered in a matrix of order 100 * 1000. These files show the pattern in which the numbers are being produced.
The 5 files 'sorted_dataset_to_count1.txt', 'sorted_dataset_to_count2.txt', ... consists of the same 100 * 1000 matrix in which the numbers in a line are sorted for easy counting of the numbers.
